### Values

Values in Javascript are something of the form shown below.

    - 42 : number
    - 3.14 : number
    - "Hello world" : string
    - true : boolean
    - false : boolean
    - null
    - undefined
    - [1,2,3,5,7]
    - {name : "Sourav"}
