### Values

Values in Javascript are something of the form shown below.

    - 42 : number
    - 3.14 : number
    - "Hello world" : string
    - true : boolean
    - false : boolean
    - null : empty values
    - undefined : empty values
    - [1,2,3,5,7] : non-primitive values (array is an ordered set of list)
    - {name : "Sourav"} : Here unlike position 0 or 1, positions are given names known as *key*
